Tentang Saya

Website ini dibuat sebagai proyek Ujian Akhir Semester (UAS) Program Studi Sistem Informasi Semester 1. Tujuan utamanya adalah untuk mendemonstrasikan kemampuan teknis dalam merancang antarmuka web yang responsif, modern, dan informatif. Setiap elemen di sini dirancang untuk menunjukkan perkembangan saya selama satu semester pertama ini.

Saya memiliki minat besar pada pengembangan teknologi informasi dan analisis sistem. Melalui mata kuliah Pemrograman Web, saya belajar bahwa sebuah website bukan hanya sekadar kode, tetapi juga tentang bagaimana memberikan pengalaman pengguna (user experience) yang nyaman dan fungsional.

Website ini juga menjadi sarana dokumentasi pembelajaran yang akan terus saya kembangkan di semester berikutnya. Ini adalah langkah awal saya untuk menjadi seorang profesional di bidang Sistem Informasi.
